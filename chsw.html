¿Cómo hacer un servidor web?

La tarea de crear un servidor requiere de conocimientos profesionales de programación para lograr un sistema funcional a lo largo del tiempo, además de un hardware y software que cuente con las características necesarias para dar el soporte adecuado. Asimismo, existe la posibilidad de hacer un servidor web casero, para ello requieres lo siguiente:

  1. Comprueba que el IP de tu portátil es fijo. Luego, ingresa y crea una cuenta con un correo electrónico a NOIP, allí podrás crear el nombre del dominio (hostname) para tu IP.


  2. Utiliza todas las herramientas que incluye XAMPP para crearlo. Linux será el sistema operativo; Apache, el servidor web gratuito; MySQL o MariaDB serán la base de datos; PHP, el lenguaje de programación; y Perl, el lenguaje para administrar el sistema.


  3. Configura el router desde tu portátil para que reciba y envíe datos al mismo puerto que corresponda al servidor web.

  4. Finalmente, debes configurar el firewall de tu portátil para que dé paso al tráfico en el puerto que pertenece al servidor web.
